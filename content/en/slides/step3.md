### Метод Эйлера

```python	
	function [x,t]=eiler_m(a,b,n,x0) 
	  tic(); 
	  h=(b-a)/n;
	  x(1)=x0;
	  for i=1:n+1
	    t(i)=a+(i-1)*h;
	  end
	  for i=2:n+1
	    tp=t(i-1)+h/2;
	    xp=x(i-1)+h/2*g(t(i-1),x(i-1));
	    x(i)=x(i-1)+h*g(tp,xp);
	  end
	  established_time = toc()
	end
```

### Метод Рунге-Кутта

```python	
	function [x,t]=runge_kut(a,b,n,x0)
	  tic();
	  h=(b-a)/n;
	  x(1)=x0;
	  for i=1:n+1
	    t(i)=a+(i-1)*h;
	  end
	  for i=2:n+1
	    K1=g(t(i-1),x(i-1));
	    K2=g(t(i-1)+h/2,x(i-1)+h/2*K1);
	    K3=g(t(i-1)+h/2,x(i-1)+h/2*K2);
	    K4=g(t(i-1)+h,x(i-1)+h*K3);
	    delt=h/6*(K1+2*K2+2*K3+K4);
	    x(i)=x(i-1)+delt;
	  end
	  established_time = toc()
	end
```
